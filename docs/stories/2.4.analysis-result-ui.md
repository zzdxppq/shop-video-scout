# Story 2.4: 镜头分析结果页面

## Status
Draft

## Story

**As a** 用户,
**I want** 查看AI分析后的镜头分类和推荐结果,
**so that** 可以了解素材质量并确认进入下一步

## Acceptance Criteria

### AC1: 分析进度展示

**Scenario**
```gherkin
GIVEN 用户触发AI分析
WHEN 分析进行中
THEN
  - 显示整体进度百分比
  - 显示"正在分析第N/M个视频"
  - 分析完成后自动跳转到结果页
```

**UI Interaction**
| Trigger | Behavior |
|---------|----------|
| 分析开始 | 显示进度条和状态文字 |
| 分析完成 | 自动展示结果 |

---

### AC2: 镜头分类展示

**Scenario**
```gherkin
GIVEN 分析完成
WHEN 用户查看结果
THEN
  - 按分类Tab展示（食物/人物/环境/其他）
  - 每个镜头显示缩略图和评分
  - 推荐镜头显示特殊标记
```

**Business Rules**
| ID | Rule |
|----|------|
| BR-2.1 | 推荐镜头显示星标 |
| BR-2.2 | 默认展示"全部"Tab |
