# Story 2.2: 前端视频上传组件

## Status
Draft

## Story

**As a** 用户,
**I want** 通过拖拽或点击上传视频文件,
**so that** 可以方便地批量上传探店素材

## Acceptance Criteria

### AC1: 拖拽上传区域

**Scenario**
```gherkin
GIVEN 用户在创建任务页面
WHEN 用户拖拽视频文件到上传区域
THEN
  - 显示拖拽高亮效果
  - 释放后开始上传
  - 显示上传进度
  - 上传完成后显示缩略图
```

**Business Rules**
| ID | Rule |
|----|------|
| BR-1.1 | 支持同时拖拽多个文件 |
| BR-1.2 | 自动过滤非视频文件 |
| BR-1.3 | 显示已上传数量/最大数量 |

**UI Interaction**
| Trigger | Behavior |
|---------|----------|
| 拖拽进入 | 高亮上传区域，显示"释放上传" |
| 上传中 | 显示进度条和取消按钮 |
| 上传完成 | 显示缩略图和删除按钮 |

---

### AC2: 视频列表管理

**Scenario**
```gherkin
GIVEN 已上传视频
WHEN 用户管理视频列表
THEN
  - 网格展示已上传视频缩略图
  - 支持删除单个视频
  - 显示视频时长
```
