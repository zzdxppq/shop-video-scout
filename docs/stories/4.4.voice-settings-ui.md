# Story 4.4: 配音设置页面

## Status
Draft

## Story

**As a** 用户,
**I want** 选择配音音色并预览效果,
**so that** 可以选择满意的配音风格

## Acceptance Criteria

### AC1: 音色选择

**Scenario**
```gherkin
GIVEN 用户进入配音设置页面
WHEN 用户选择音色
THEN
  - 显示标准音色列表（活泼女声、阳光男声、知性女声）
  - 如有克隆声音，显示"我的声音"选项
  - 支持试听示例音频
```

**Business Rules**
| ID | Rule |
|----|------|
| BR-1.1 | 默认选中"活泼女声" |
| BR-1.2 | 试听示例时长5-10秒 |

**UI Interaction**
| Trigger | Behavior |
|---------|----------|
| 点击音色卡片 | 选中并播放试听 |
| 点击试听按钮 | 播放示例音频 |

---

### AC2: 开始合成

**Scenario**
```gherkin
GIVEN 用户选择了音色
WHEN 用户点击开始合成
THEN
  - 显示确认弹窗
  - 确认后触发配音合成
  - 显示合成进度页面
```
