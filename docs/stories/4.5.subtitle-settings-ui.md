# Story 4.5: 字幕设置页面

## Status
Draft

## Story

**As a** 用户,
**I want** 设置视频字幕样式,
**so that** 可以选择喜欢的字幕效果

## Acceptance Criteria

### AC1: 字幕开关和样式选择

**Scenario**
```gherkin
GIVEN 用户在配音设置页面
WHEN 用户设置字幕选项
THEN
  - 显示字幕开关（默认开启）
  - 显示5种字幕样式预览
  - 选择样式后显示效果预览
```

**Business Rules**
| ID | Rule |
|----|------|
| BR-1.1 | 字幕默认开启 |
| BR-1.2 | 5种预设样式：简约白字、活力黄字、小红书风、抖音热门、霓虹炫彩 |

**UI Interaction**
| Trigger | Behavior |
|---------|----------|
| 点击字幕开关 | 切换字幕启用状态 |
| 点击样式卡片 | 选中并显示预览效果 |
