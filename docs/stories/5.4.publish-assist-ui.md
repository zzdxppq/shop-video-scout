# Story 5.4: 发布辅助UI组件

## Status
Draft

## Story

**As a** 用户,
**I want** 查看和复制推荐的话题和标题,
**so that** 可以方便地发布视频到社交平台

## Acceptance Criteria

### AC1: 话题标签展示与复制

**Scenario**
```gherkin
GIVEN 用户在结果页面
WHEN 查看话题推荐
THEN
  - 显示话题标签列表
  - 支持一键复制所有话题
  - 支持单个话题复制
```

**Business Rules**
| ID | Rule |
|----|------|
| BR-1.1 | 话题标签显示为芯片样式 |
| BR-1.2 | 复制后显示成功提示 |

**UI Interaction**
| Trigger | Behavior |
|---------|----------|
| 点击一键复制 | 复制所有话题到剪贴板，显示成功Toast |
| 点击单个话题 | 复制该话题，显示成功提示 |

---

### AC2: 标题推荐展示与复制

**Scenario**
```gherkin
GIVEN 用户在结果页面
WHEN 查看标题推荐
THEN
  - 显示标题选项列表
  - 每个标题带复制按钮
```

**Business Rules**
| ID | Rule |
|----|------|
| BR-2.1 | 标题显示为单选列表样式 |

**UI Interaction**
| Trigger | Behavior |
|---------|----------|
| 点击复制按钮 | 复制对应标题，显示成功提示 |
